# tools

A collection of small, self-contained utilities for everyday developer workflows.

## Tools

| Tool | Description | Platform |
|---|---|---|
| [claude-usage](./claude-usage/) | macOS menu bar widget showing Claude Pro 5hr window and weekly usage | macOS |

---

## Philosophy

Each tool in this repo:
- Does one thing
- Has no external runtime dependencies where possible
- Includes a one-command installer
- Works without any cloud account or third-party service beyond what it's measuring
